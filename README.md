# SOS Local — Central de Emergência em Angola

Plataforma web de utilidade pública que centraliza os contactos oficiais de emergência em Angola, permitindo que residentes, turistas e visitantes encontrem rapidamente o número certo em qualquer situação.

## Serviços disponíveis

| Serviço | Número |
|---|---|
| Polícia Nacional | 113 |
| Emergência Médica | 112 |
| Corpo de Bombeiros | 115 |
| Farmácias de Plantão | — |

## Tecnologias utilizadas

- HTML5 semântico
- CSS3 com variáveis (design system)
- JavaScript / jQuery 3.7.1
- Leaflet.js + OpenStreetMap (mapa interativo)
- Dark mode com `prefers-color-scheme`
- Totalmente responsivo (mobile, tablet, desktop)

## Funcionalidades

- Menu responsivo com hamburger
- Dark mode persistente
- Botão "Voltar ao topo"
- Mapa interativo com geolocalização
- Carrossel acessível (mobile)
- Validação de formulário com funções JavaScript puras
